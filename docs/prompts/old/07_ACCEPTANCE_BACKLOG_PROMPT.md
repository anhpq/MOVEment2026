# Prompt 07 - Tạo backlog và tiêu chí nghiệm thu

Bạn là senior technical lead. Hãy tổng hợp các phân tích trước đó thành backlog phát triển MOVEment 2026.

## Context

Các nhóm phân tích đã có:

- Product scope
- Player screens
- Station QR flow
- Admin functions
- Database/API
- UI reference

## Nhiệm vụ

1. Chia backlog theo epic:
   - Player UI
   - QR/check-in/check-out flow
   - Scoring
   - Leaderboard
   - Admin
   - Database/API
   - Auth/RBAC
   - Logs/audit
   - UI polish
   - Testing
2. Với mỗi user story, ghi:
   - title
   - user story
   - priority: P0/P1/P2
   - acceptance criteria
   - dependencies
   - API/data needed
   - test notes
3. Xác định MVP launch checklist.
4. Xác định các câu hỏi cần quyết định trước khi code.
5. Đưa ra thứ tự làm việc khuyến nghị.

## Output mong muốn

Trả về Markdown gồm:

- Epic List
- User Stories
- MVP Checklist
- Decision Log Needed
- Recommended Build Order
- Test Plan Summary

Không viết code ở bước này.
